## Installation

### Prerequisites

* Node.js (Latest Version Recommended)
* npm

### Clone Repository

```bash
git clone https://github.com/krishnakonakala/krishna-teja-portfolioo.git
```

### Navigate to Project

```bash
cd krishna-teja-portfolioo
```

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```
